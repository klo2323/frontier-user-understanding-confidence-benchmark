# Evaluations

This folder stores model-agnostic evaluation assets for Kelsey Ontko's turn-by-turn confidence scoring engine.

## Folders

- `scenarios/`: reusable evaluation scenarios.
- `runs/`: raw model/scorer run outputs by provider, model, and date.
- `results/`: aggregated metrics and comparison tables.

The MVP keeps evaluations as plain files so the benchmark remains easy to inspect, reproduce, and publish on GitHub.
